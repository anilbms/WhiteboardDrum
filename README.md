WhiteboardDrum
==============

Rhythm machine playing the rhythm pattern on a whiteboard.
WebCam, getUserMedia() and WebAudioAPI ready browser is needed.

Live Demo ia available at:  
http://www.g200kg.com/whiteboarddrum/
